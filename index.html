Melancholy Pocket

Go To Website
Year2024
RoleWeb Designer · UX Developer
ToolsWix · Figma · HTML / CSS
TypeInteractive Documentary · TAU Project

An interactive documentary project aimed at archiving the old Israeli Broadcast Television building in Jerusalem through an interactive web design. This project deals with the history of the building and what was accomplished in it before it was destroyed. Documenting how the building’s memory lives on.

Process

01
Research & Narrative Structure

Researched the building's history and mapped the stories, artifacts, and personal accounts forming the documentary's backbone. Each section tied to a specific memory.

02
UX Wireframing

Translated the documentary structure into user flows and wireframes. Designed an interface allowing visitors to drift between spaces, surfacing archival material through interaction rather than passive scrolling.

03
Visual Design

Developed a visual language echoing Polly Pocket toy houses — purple and pink tones, smooth textures reflecting the accumulation of time and memory.

04
Development & Launch

Built and deployed on Wix with custom CSS and interactive layers. Publicly accessible — making this piece of broadcast history available to all.

ToolsFigma
TypeUX Prototype · Mobile App

A figma prototype exploring easy to grow and care for plants. For the average curious beginner, this prototype is designed to inspire and guide users on their green journey. Helps users discover plants, build confidence through guided routines, and develop lasting care habits.

05

Dorothy

Play Dorothy
Year2023
RoleDeveloper · Designer
ToolsHTML · CSS · JavaScript
TypeBrowser Game · TAU Gamification Course

A browser game built from scratch for a TAU gamification course. Dorothy inverts the typical action-game win condition — the player must guide the character with sensitivity and care, not aggression.

Process

01
Concept & Gamification Theory

Grounded in gamification theory: aggression and speed are penalised; patience and care are the only path to winning. Translated academic literature on empathy-based game design into a playable form.

02
Design

Designed all visual assets with a deliberately soft, warm aesthetic — colour, typography, and character design all reinforcing care over conflict.

03
Development

Built the complete game in vanilla HTML, CSS, and JavaScript — no frameworks. Every mechanic was intentional, suited to a project where mechanics do philosophical work.

Print & Infographics

Print design spanning brochures and infographic publications — translating complex information into coherent layouts where every design decision is permanent.

Approach

01
Content Architecture

Mapped the information hierarchy before any visual design — what the reader needs first, second, and in what depth. Every layer placed in the correct order before the grid was built.

02
Grid & Typography

Established a precise grid system for each piece, selecting typefaces for print rendering specifically — what reads well on screen often requires adjustment for paper and ink.

03
Print Preparation

Prepared all files for professional print — bleed, crop marks, CMYK colour profiles, embedded fonts, and resolution — ensuring screen designs printed faithfully.

Poster Design

Poster design for film and festival contexts — distilling a complete story into a single arresting visual moment. Composition, typographic hierarchy, and negative space all used toward that singular goal.

Approach

01
Film / Event Research & Concept

Closely studied each film or event — genre, tone, characters, central conflict — identifying the single image or tension the entire work orbits around.

02
Composition & Typography

03
Refinement

Iterated on colour, contrast, and image treatment until emotionally resonant.
